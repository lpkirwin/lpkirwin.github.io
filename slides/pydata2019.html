Basic encoding strategies

Label encoding

Simple 1-for-1 replacement, but implies ordinal relationship (!)


						["fuzzy", "smooth", "soft", "smooth"] -> [0, 1, 2, 1]

One-hot encoding

(AKA dummy/indicator/binary encoding.) Standard approach for most algorithms, but high cardinality will create huge number of features.


						["fuzzy", "smooth", "soft", "smooth"] -> [[1, 0, 0],
						                                          [0, 1, 0],
						                                          [0, 0, 1],
